Ayuda de MotoGP-Desktop

Inicio

La pagina indice accede a la portada del proyecto. Desde aquí puedes navegar al resto de secciones mediante el menú superior y consultar atajos a los contenidos más recientes del curso.

Piloto

La pagina piloto contien la ficha de Francesco "Pecco" Bagnaia piloto de MotoGP 2025. La página muestra nombre y dorsal, foto, equipo y fabricante, nacionalidad, breve biografía, principales estadísticas de la última temporada. Incluye también un resumen de resultados destacados y una galería básica de imágenes del piloto. La pagina también contiene un glosario con terminos de MotoGP.

Circuito

La pagina circuito muestra tres opciones para insertar diferentes archivos. El primer input solicita un archivo de tipo html, una vez seleccionado este se mostara en la pagina. El segundo input solicita un archivo con la extension kml. Una vez seleccionado se mostrara un mapa dinamico en la pagian. Por ultimo, el tercer input solitica un archivo con la extension svg. Se mostrara el svg seleccionado por el usuario en la pagina.

Meteorología

La pagina meteorologia contiene informacion del lugar donde se celebrara el Gran Premio de las Americas: Ciudad, pais, población, etc. Esta tambien mostrara información relacionada con la meteorologia en los horarios de los entrenamientos y de la carrera.

Clasificaciones

En la pagina clasificaciones muestra el Ganador del Gran Premio de las Americas de motoGP de 2025, asi como la clasificación del mundial tras la carrera.

Juegos

La pagina juegos contiene enlaces a los diferentes juegos.

Cronometro

En cronometro encontrarás un cronómetro para medir intervalos de tiempo. La lectura se muestra en formato mm:ss.s y se actualiza cada décima de segundo. Dispone de tres botones: “Arrancar” inicia o reanuda la medición, “Parar” la detiene sin perder el tiempo acumulado y “Reiniciar” pone el contador a cero mostrando 00:00.0.

Cronometro php

En cronometroPHP encontrarás un cronómetro para medir intervalos de tiempo. Dispone de tres botones: “Arrancar” inicia o reanuda la medición, “Parar” la detiene sin perder el tiempo acumulado y “Mostar” muesta el tiempo medido.

Memoria

En memoria encontrarás un juego de cartas por parejas cuyo objetivo es descubrir todas las parejas en el menor tiempo posible. Voltea cartas de dos en dos: si las imágenes coinciden, ambas quedan reveladas y dejan de responder; si no coinciden, se pondrán de nuevo bocabajo tras un breve retardo, durante el cual el tablero queda bloqueado para evitar pulsaciones accidentales. El cronómetro, situado bajo el título del juego, muestra el tiempo en formato mm:ss.s, se inicia automáticamente al comenzar la partida y se detiene cuando todas las parejas han sido descubiertas.

Administración de la base de datos

En Administración de la base de datos encontrarás tres botones para gestionar la base de datos de las pruebas de usabilidad. El botón “Reiniciar base de datos” crea la base de datos si no existe y, en caso de existir, la reinicia a su estado inicial. El botón “Eliminar base de datos” elimina la base de datos por completo. Por último, el botón “Exportar datos a CSV” exporta el contenido de la base de datos en un archivo con formato CSV.

Pruebas de usabilidad

En Pruebas de usabilidad encontrarás un botón que dice “Iniciar prueba”. Una vez pulsado, se mostrará una serie de preguntas con un espacio en blanco en el que se podrá escribir y responder cada una. Una vez respondidas todas las preguntas, se deberá pulsar el botón situado debajo. A continuación, se mostrarán otros campos que deberán ser rellenados tanto por el usuario como por el observador presente en la prueba.
